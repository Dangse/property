## 1. 상속세 (상속세 및 증여세법)

### 1.1 시행 현황 — 2024 정부개정안 부결

2024.7.25 기획재정부가 발표한 상속세 개편안(최고세율 50%→40%, 자녀공제 5천만→5억)은
**2024.12 국회 본회의에서 부결**(찬성 98 / 반대 180 / 기권 3)되었습니다.

→ **2026.5 현재 적용 중인 상속세는 종전 그대로(50% 최고세율, 자녀공제 5천만)** 입니다.

- 출처: [매거진한경 — '상속세 최고세율 50→40% 인하' 불발](https://magazine.hankyung.com/business/article/202412108212b)
- 출처: [뉴스1 — 상속세 최고세율 50% 유지…자녀공제 1인당 5000만원까지만](https://www.news1.kr/economy/trend/5627677)

향후 재개정이 통과되면 `src/core/constants.js`의 `INHERIT_*` 상수와
`src/core/inheritance-tax.js`의 세율표만 교체하면 됩니다.

### 1.2 누진세율표 (상증법 §26)

| 과세표준 구간 | 세율 | 누진공제 |
|---|---|---|
| 1억 이하 | 10% | 0 |
| 1억 초과 ~ 5억 이하 | 20% | 1,000만 |
| 5억 초과 ~ 10억 이하 | 30% | 6,000만 |
| 10억 초과 ~ 30억 이하 | 40% | 1억 6,000만 |
| 30억 초과 | 50% | 4억 6,000만 |

- 출처: [국세청 — 상속세 세율](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6529&cntntsId=7957)

### 1.3 상속공제 (상증법 §18~§24)

| 공제 종류 | 금액 |
|---|---|
| 기초공제 | 2억 |
| 자녀공제 | 1인당 5,000만 |
| 미성년자공제 | 1,000만 × 19세까지 잔여연수(미성년자 1인당) |
| 연로자공제 (65세 이상) | 1인당 5,000만 |
| 장애인공제 | 1,000만 × 기대여명연수 |
| 일괄공제 | 5억 (기초+인적공제 합계와 비교해 큰 금액 선택) |
| 배우자공제 | 최소 5억 ~ 최대 30억 (법정상속분 한도) |
| 동거주택상속공제 | 6억 한도 (주택가액 − 채무) |
| 금융재산상속공제 | 순금융재산 2천만 이하: 전액 / 초과: 20%, 최소 2천만, 최대 2억 |

- 출처: [국세청 — 상속공제 항목별 설명](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6528&cntntsId=7956)

### 1.4 배우자상속공제 (상증법 §19)

배우자공제 한도 = min(
  배우자 법정상속분 × 상속재산 − 배우자 사전증여재산공제 초과액,
  30억
)
- 최소 5억 보장 (배우자가 실제 상속받지 않거나 받은 금액이 5억 미만이어도 5억 공제)
- 법정상속분: 배우자 1.5 / 자녀 1.0 → 자녀 n명일 때 배우자 = 1.5 / (n + 1.5)

### 1.5 세대생략 할증과세 (상증법 §27)

상속인·수유자가 피상속인의 자녀가 아닌 직계비속(예: 손자녀)인 경우:
- 기본 할증 30%
- 미성년자 + 상속재산 20억 초과: 40%

- 출처: [국세청 — 세대생략 할증세액](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6530&cntntsId=7958)

### 1.6 단기재상속 세액공제 (상증법 §30)

10년 이내 재상속 시 직전 상속세 중 재상속 재산 해당분을 다음 비율로 공제:

| 경과 연수 | 공제율 |
|---|---|
| 1년 이내 | 100% |
| 2년 이내 | 90% |
| 3년 이내 | 80% |
| ... | (1년당 10%p 감소) |
| 10년 이내 | 10% |

- 출처: [상증법 §30 — LBOX 법령](https://lbox.kr/v2/statute/%EC%83%81%EC%86%8D%EC%84%B8%EB%B0%8F%EC%A6%9D%EC%97%AC%EC%84%B8%EB%B2%95/%EB%B3%B8%EB%AC%B8%20%3E%20%EC%A0%9C2%EC%9E%A5%20%3E%20%EC%A0%9C7%EC%A0%88%20%3E%20%EC%A0%9C30%EC%A1%B0)

### 1.7 신고세액공제 (상증법 §69)

산출세액(가산세 등 제외) × 3%

### 1.8 간주상속재산 (상증법 §8~§10)
- 보험금 (피상속인이 보험료를 실질 부담한 보험)
- 신탁재산
- 퇴직금·퇴직수당·공로금 등

### 1.9 추정상속재산 (상증법 §15)
사망일 전 다음 처분·인출·채무부담 금액 중 용도불분명 부분을 상속재산에 가산:
- **1년 이내 2억 이상**
- **2년 이내 5억 이상**

- 출처: [국세청 — 추정상속재산](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6523&cntntsId=7951)

### 1.10 사전증여재산 합산 (상증법 §13)
- 상속인: 사망일 전 **10년** 이내 증여재산
- 비상속인(예: 손자녀): 사망일 전 **5년** 이내 증여재산
- 기납부 증여세는 증여세액공제(§28)로 차감

### 1.11 장례비 공제 (상증법 §14①2, 시행령 §9의2)
- 일반장례비: 500만 미만이면 500만 일률공제, 1,000만 초과 시 1,000만 한도
- 봉안시설·자연장지 사용액: 별도 500만 한도
- → 합계 최대 **1,500만**

### 1.12 감정평가수수료 (상증법 시행령 §20의3)
500만 한도

## 2. 상속 취득세 (지방세법 §11①4, §15)

| 구분 | 취득세율 | 농특세 | 교육세 |
|---|---|---|---|
| 상속 1주택 (1세대) | 0.8% | 면제 (85㎡ 이하 농특세 면제) | 0.16% |
| 상속 다주택 / 비1세대1주택 | 2.8% | 0.2% (85㎡ 초과) | 0.16% |
| 상속 농지 | 2.3% | 0.2% | 0.06% |
| 상속 일반 토지/건물 | 2.8% | 0.2% | 0.16% |

- 1주택 상속 0.8% 특례: 세대 전원이 상속개시일 현재 무주택일 것

## 3. 상속주택 주택수 제외 특례

### 3.1 종합부동산세 (종부세법 §15의2)
- 상속개시일부터 **5년간 1세대1주택자 판정 시 주택수 제외**
- 5년 경과 후에도 지분율 40% 이하 또는 지분 상당 공시가격 6억(수도권 외 3억) 이하면 계속 제외
- 신청 필요 (합산배제 신고)

### 3.2 재산세 (지방세법 §111의2)
- 1세대1주택자 판정 시 동일한 5년 특례 적용

### 3.3 양도소득세 (소득세법 §89, 시행령 §155②)
- 일반주택 + 상속주택(피상속인이 사망일 현재 보유한 주택)을 동시 보유 시
- **일반주택 양도 시 상속주택을 주택수에서 제외하여 1세대1주택 비과세 적용 가능**
- 12억 비과세 한도 적용
- 출처: [국세청 — 1세대 1주택자 판단 시 주택 수 산정 제외 특례](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=40621&cntntsId=239014)

## 4. 상속 후 양도 — 취득시기·취득가액 의제

### 4.1 6개월 이내 양도
- 양도가액 = 취득가액 (상속재산가액으로 평가됨)
- → 양도차익 0, 양도세 0

### 4.2 보유기간 기산 (1세대1주택 비과세 판정용)
- 피상속인과 **동일세대원**이 상속받은 경우: 피상속인 보유·거주기간 통산
- 별도세대원: 상속개시일부터 기산

### 4.3 12억 초과 부분 과세 (소득세법 §89②)
- 출처: [흠택스 — 상속주택 양도세 비과세 요건](https://www.heumtax.com/contents/posts/inherited-home-tax-exemption)

## 5. 종부세 중과 폐지 (2022 세법개정 이후)

- 3주택 이상 + 과세표준 12억 초과 구간만 차등세율(2.0%~5.0%)
- 그 외 구간은 1·2주택과 동일세율(0.5%~2.7%)
- 조정대상지역 2주택 중과는 폐지됨
- 출처: [삼일PwC — 다주택소유자 세금](https://www.pwc.com/kr/ko/insights/issue-brief/one-point-tax-05.html)

## 6. 보수적 정책 결정 사항

1. **자녀공제는 5천만 적용** (개정안 부결로 5억 미적용)
2. **최고세율 50% 유지**
3. **상속주택 5년 특례는 사용자 옵션으로 분리** — 신청 여부를 입력값으로 받음
4. **간주·추정상속재산은 입력 필드로 노출** (자동 추정 미수행)
5. 향후 법령 개정 시 본 문서를 갱신하고, `src/core/constants.js`의 관련 상수만 교체
